# Web Scraping

## Project 1: Quotes Scraper
Experimenting with web scraping using BeautifulSoup
Scrapes all 100 quotes from quotes.toscrape.com using BeautifulSoup and Requests.
Results exported to a timestamped CSV using Pandas (or txt file that has been commented) or displayed on the screen based on user input.

### Tech Stack
- Python 3.12
- BeautifulSoup4
- Requests
- Pandas

### How to Run
1. Create virtual environment: `python3 -m venv .venv`
2. Activate: `source .venv/bin/activate`
3. Install dependencies: `pip install -r requirements.txt`
4. Run: `python3 webScrapeWithPagination.py`
